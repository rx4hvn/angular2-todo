# Angular2-todo
Very simple To-Do application buit on MEAN.

[Live Demo](https://todo2-rx4hvn.herokuapp.com/) on heroku


## How to use
    git clone https://github.com/rx4hvn/angular2-todo.git
    npm install
    node server.js
    
## Libraries used
- Node.js
- MongoDB [mLab](https://mlab.com/)
- Express
- Angular2
